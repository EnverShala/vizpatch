## Wichtige Eigenschaft: Kein Test wurde ausgeführt

Dieser Plan liefert ausschließlich das **Test-Artefakt-Paket**. Die eigentliche Ausführung
des Tests ist ein **separater manueller Halb-Tag-Termin** (D-18), der von Vizionists
durchgeführt wird — voraussichtlich vor dem Vor-Ort-Termin beim Tankstellen-Kunden.

Zeitplan für die Test-Ausführung:
1. Vizionists liest `PRE-DEPLOYMENT-TEST.md` durch (~10 Min Vorbereitung)
2. Docker-Image bauen + Container starten (~20 Min)
3. Einzel-Mails testen + Prompt-Iteration (~60-90 Min)
4. Multi-Turn-Test (~40-60 Min)
5. Reboot-Test + Report (~30 Min)
6. Gesamtaufwand: ~2.5-3.5 h

Das Ergebnis des Tests wird in `PRE-DEPLOYMENT-TEST-REPORT.md` dokumentiert.
Nach Freigabe (≥8/10 Klassifikation, ≥3.5/5 Qualität, alle Reboot-Checks) kann
das Deployment-Paket (aus Plan 02.04) zum Kundentermin mitgenommen werden.
